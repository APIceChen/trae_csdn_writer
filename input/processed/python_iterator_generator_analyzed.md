### **关键知识点**

#### 知识点 1：for 循环的本质
```
for 循环 = iter() 获取迭代器 + 循环调用 next() 取值 + StopIteration 异常处理
```

#### 知识点 2：可迭代对象 vs 迭代器
```python
# 可Iterable对象：有 __iter__() 方法
# 迭代器：有 __iter__() 和 __next__() 方法
# 关系：iter(可迭代对象) → 得到迭代器
```

#### 知识点 3：迭代器的状态性
> 迭代器是有状态的，会记住当前遍历位置

#### 知识点 4：yield 的魔法
- `yield` 不仅返回值，还**暂停函数执行状态**
- 下次 `next()` 从暂停位置继续
- 生成器是**特殊的迭代器**

#### 知识点 5：生成器的优势
- **按需生成**数据，节省内存
- 适合**大数据量**或**延迟计算**场景
