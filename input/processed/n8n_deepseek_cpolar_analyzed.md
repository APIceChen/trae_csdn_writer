## 💡 关键技术要点提取

### **Docker 部署关键参数**

```bash
# 必须配置项
N8N_BASIC_AUTH_ACTIVE=true    # 启用基础认证
N8N_BASIC_AUTH_USER=admin      # 管理员用户名
N8N_BASIC_AUTH_PASSWORD=xxx    # 强密码（必须修改！）
TZ=Asia/Shanghai              # 中国时区
N8N_COOKIE_SECURE=false        # 禁用安全Cookie（重要！）
restart always                 # 开机自启+异常重启
```

### **Docker 镜像加速器配置**

```json
{
    "registry-mirrors": [
        "https://docker.xuanyuan.me",
        "https://docker.m.daocloud.io"
    ]
}
```

**重要性**：🔴 **必做步骤**，否则拉取镜像会失败或极慢！

### **DeepSeek API 接入要点**

1. 免费额度适合学习和测试
2. 密钥只显示一次，必须立即保存
3. 在 n8n 中选择 "DeepSeek" 作为凭证类型

### **cpolar 隧道配置关键**

```yaml
协议: http
本地端口: 5678  # n8n 默认端口
地区: China Top # 选择国内节点速度更快
域名类型: 二级子域名（固定） > 随机域名
```
